# Rice Ball POS System 🍙

## Features
- POS system (order management)
- Dashboard (sales analytics)
- AI chatbot (via n8n)

## Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: Supabase (PostgreSQL)
- AI Workflow: n8n

## Setup
1. Clone repo
2. Add .env 
3. Run frontend/backend
